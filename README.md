# uk-station-data

This is a JS/TS library containing data about all[^1] UK train stations and lines. Currently, this library is only used for [Stationary](https://github.com/TangledWiresOfficial/Stationary).

## Installation
You can add this library to your project with this command:
```shell
npm add @tangledwires/uk-station-data
```

[^1]: This library is not complete yet.
